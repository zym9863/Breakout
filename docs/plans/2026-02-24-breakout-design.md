# Breakout 游戏设计文档

## 概述

基于 Svelte + Vite + Canvas 的豪华版 Breakout 打砖块游戏，新海诚式忧郁清新视觉风格。

## 技术栈

- **框架**: Svelte + Vite（纯静态 SPA）
- **渲染**: HTML5 Canvas 2D
- **语言**: TypeScript
- **状态管理**: Svelte stores
- **音频**: Web Audio API（合成音效）
- **持久化**: localStorage（排行榜 + 自定义关卡）

## 核心架构

### 游戏循环
- `requestAnimationFrame` 驱动
- 固定时间步长更新物理
- Canvas 2D 渲染

### 状态管理
游戏状态：`menu | playing | paused | gameover | editor | leaderboard`
通过 Svelte store 管理页面切换。

### 碰撞检测
- AABB 矩形碰撞 + 球的圆形碰撞修正
- 球碰挡板时根据碰撞位置调整反射角度（越靠边角度越大）

### 操控方式
同时支持鼠标和键盘（方向键/AD 键）控制挡板。

## 视觉设计 — 新海诚风格

### 配色
- **背景**: 天空渐变 `#87CEEB` → `#FFB6A3`（黄昏天空）
- **砖块**: 柔和粉、紫、蓝、金色系，半透明 + 发光
- **挡板**: 深色调，柔和边缘光晕
- **球**: 白色核心 + 光晕拖尾

### 粒子特效
- 砖块碎裂：对应颜色的光粒子飘散
- 球尾迹：淡光点轨迹
- 道具下落：闪烁光斑
- 背景粒子：缓慢漂浮光斑/尘埃
- 过关特效：全屏光芒绽放

### 渲染层次
1. 天空渐变背景
2. 背景漂浮粒子
3. 砖块（带光晕）
4. 道具
5. 球 + 尾迹
6. 挡板
7. 前景粒子特效
8. HUD 叠加层

## 游戏玩法

### 砖块类型
| 类型 | 行为 | 得分 |
|------|------|------|
| 普通砖 | 1 次击碎 | 10 |
| 硬砖 | 2-3 次击打 | 20-30 |
| 金砖 | 击碎必掉道具 | 50 |
| 不碎砖 | 无法击碎 | — |

### 道具系统
| 道具 | 效果 | 持续时间 |
|------|------|----------|
| 加宽挡板 | 宽度 x1.5 | 15 秒 |
| 多球 | 分裂为 3 个 | 直到额外球落下 |
| 减速 | 球速 -30% | 10 秒 |
| 穿透球 | 穿过砖块不反弹 | 8 秒 |
| 额外生命 | +1 生命 | 永久 |
| 磁力挡板 | 球粘在挡板上 | 下一次接球 |

### 关卡
5 个预设关卡，难度递进：
1. 纯普通砖
2. 加入硬砖
3. 加入不碎砖
4. 金砖 + 密集排列
5. 混合高难度

### 计分
- 3 条初始生命
- 连击倍率：连续击砖不碰挡板，得分 x2/x3/...

## 关卡编辑器
- 网格式编辑界面（Svelte 组件）
- 点击切换砖块类型
- 导出/导入 JSON（localStorage）
- 可直接试玩自制关卡

## 音效
- Web Audio API 合成音效
- 音效：碰挡板、碰砖块、碰墙壁、碎裂、道具拾取、失去生命、过关、Game Over
- 无背景音乐

## 排行榜
- localStorage 存储
- 记录：玩家名、分数、达到关卡、日期
- 前 10 名，分数降序

## 页面流程
```
主菜单 → 开始游戏 → 游戏中 ⇄ 暂停
                    ↓ Game Over → 排行榜输入 → 主菜单
                    ↓ 通关 → 排行榜输入 → 主菜单
主菜单 → 关卡编辑器 → 试玩
主菜单 → 排行榜 → 返回
```
